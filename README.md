## Deployment
<details>
 <Summary> Click to See the Deployment</Summary>
This project is ready to be deployed using GitHub Pages.

Go to your repository on GitHub.

Click on Settings.

On the left sidebar, click on Pages.

Under Build and deployment, select Deploy from a branch.

Under Branch, select main (or master) and / (root) folder.

Click Save.

Your site will be live in a few minutes at https://<your-username>.github.io/<repository-name>/.
